📘 Official AWS Documentation (All Services)

🔗 AWS Docs Home
https://docs.aws.amazon.com

This is the master index of documentation for all AWS services.

📌 AWS CLI (Official Docs)
AWS CLI User Guide

🔗 https://docs.aws.amazon.com/cli/latest/userguide/

This covers:

Installing the AWS CLI (Linux, macOS, Windows)

Configuring credentials & profiles

Using commands for EC2, S3, Lambda, IAM, and more

Pagination

Output formatting

Scripting patterns

It’s the go-to reference for everything CLI.

📌 AWS CLI Command Reference

🔗 https://docs.aws.amazon.com/cli/latest/reference/

This is where you look up every single AWS CLI command and its options. For example:

aws ec2 run-instances

aws s3 cp

aws logs filter-log-events

Each command page includes flags, examples, and expected output.

Always up to date

Directly from AWS product teams

Contains examples + best practices

Covers security warnings and detailed flags

Unofficial tutorials are useful for learning, but official docs are the source of truth.
